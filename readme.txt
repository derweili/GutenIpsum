Generate blind text in Gutenberg.

== Description ==

This plugin lets you create Lorem Ipsum text inside Gutenberg.

You can generate single words, sencentes and paragraphs.

You can insert the blind text directly into Gutenberg without Copy & Paste. Or you can copy the text into you clipboard to use it anywhere.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/gutenipsum` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
4. Go to the Edit Screen of a Page or a Plugin
5. Open the GutenIpsum Sidebar via the Icon on the top Right or choose Options > GutenIpsum


== Frequently Asked Questions ==

= Does this bring a special "Lorem Ipsum Block" =

No, this plugin does not come with build in Gutenberg Blocks. It instead uses the core paragraph blocks, so everything will work even when you disable this plugin.
